# Command Line Interface
- ***Directory***

  - `.` : 현재 디렉토리

  - `..` : 현재의 상위 디렉토리 (부모 폴더)
---
- **기본 문법**

  - `touch {파일명}` : 파일 생성

  - `mkdir {디렉토리명}` : 새 디렉토리 생성

  - `ls` : 현재 작업 중인 디렉토리 내부의 폴더/파일 목록 출력
    - `ls -{}` : {}안에 -a, -al 등 옵션 추가 가능

  - `cd {디렉토리명}` : 현재 작업 중인 디렉토리를 변경

  - `start` : 폴더/파일을 열기(Mac에서는 open을 이용)

  - `rm` : 파일 삭제 (디렉토리 삭제는 -r 옵션을 추가 사용)
      - rm으로 파일 삭제 시 복구 불가능
---
- 절대 경로 : Root 디렉토리부터 목적 지점까지 거치는 모든 경로를 전부 작성한 것

- 상대 경로 : 현재 작업하고 있는 디렉토리를 기준으로 계산된 상대적 위치를 작성한 것
